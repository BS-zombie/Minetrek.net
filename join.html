Join MineTrek

Minetrek uses a custom modpack and resource pack, you need to install them to join.

We will start with forge. you can download it here:

Download Forge

Next, you will need to download the modpack and resource pack.

After you have them downloaded, you will need to place them into your games mods and resource packs folder respectively

Download Resource Pack Download Mod Pack

its also reccomended that you add additional startup flags, but these arnt strictly needed. if you dont dont know how to add them, you can skip this step.

Now you are ready to join! fire up your game and log on in!
